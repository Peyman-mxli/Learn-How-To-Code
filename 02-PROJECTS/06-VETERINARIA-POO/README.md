# 🧵 Project — Biblioteca POO
=======================================
# ✨ Overview
=============================================================================
The **Biblioteca POO** project is a library management system
built using Object-Oriented Programming (OOP) principles in Python.
This application simulates how a real library operates by managing
books, users, and borrowing processes in a structured way.
It demonstrates how real-world systems can be modeled using
clean, modular, and scalable code.
=============================================================================
🎯 Why This Project Matters
In real-world software:
Systems must manage data, relationships, and operations efficiently.
Without proper structure:
Code becomes disorganized, difficult to maintain, and hard to scale.
This project applies OOP principles to:
Organize logic, reduce complexity, and simulate real systems.
=============================================================================
📦 Core Concepts Applied
This project demonstrates:
Classes and Objects  
Encapsulation (data protection and structure)  
Inheritance (if extended)  
Separation of concerns  
Clean and modular design  
=============================================================================
📚 Features
The system includes:
Add new books to the library  
View available books  
Borrow books  
Return books  
Manage library records  
These features simulate the core operations of a real library system.

=============================================================================
⚙️ How It Works
The system follows this structure:
Libro class represents books
Usuario class represents users
Biblioteca manages operations
main.py runs the application
Users can interact with the system to:
borrow and return books while the library updates its state.
