# 🧵 Project — CLI Calculator
=======================================

# ✨ Overview
=============================================================================
The **CLI Calculator** is a command-line application built using Python
that performs basic arithmetic operations.
This project demonstrates how to build an interactive program
that accepts user input, processes logic, and displays results.
It represents a fundamental step from simple scripts
to real-world interactive applications.
=============================================================================

🎯 Why This Project Matters
In real-world development:
Applications must interact with users, validate input,
handle logic, and produce correct output.
This project introduces:
Control flow, user interaction, and structured programming.
It is a foundational project for building:
CLI tools, automation scripts, and backend logic systems.
=============================================================================
📦 Features
The application supports:
Addition
Subtraction
Multiplication
Division
It also includes:
User input handling
Basic validation
Continuous program execution (loop)
Clean output display
