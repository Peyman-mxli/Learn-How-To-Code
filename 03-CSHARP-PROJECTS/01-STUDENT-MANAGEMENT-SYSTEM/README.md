# 🧵 Student Management System

## ✨ Overview

In this project, I build a simple **Student Management System** using C#.

This project is designed to be easy to understand, clean, and beginner-friendly, but still organized like a real software project.

The goal is to practice how to manage student information inside a program.

A user will be able to add students, view students, search for students, update student information, and remove students.

This project helps me move from only learning C# concepts into actually building a complete application.

---

## 🎯 Project Goal

The main goal of this project is to create a console application that manages student records.

Each student can have information such as:

* Student ID
* Full Name
* Age
* Email
* Course Name
* Grade

This project will start simple, but it can be improved later.

In the future, I can upgrade it by adding:

* File saving
* JSON storage
* Database connection
* Login system
* Admin panel
* ASP.NET Core Web API version

## 🏗️ Project Features

This project will include the following features:

### 1. Add Student

The user can add a new student by entering student information.

Example information:

* ID
* Name
* Age
* Email
* Course
* Grade

---

### 2. View All Students

The user can see all students saved inside the system.

This helps practice lists and loops.

---

### 3. Search Student

The user can search for a student by ID or name.

This helps practice conditions and list searching.

---

### 4. Update Student

The user can update student information if something changes.

For example:

* Change name
* Change email
* Change course
* Change grade

---

### 5. Delete Student

The user can remove a student from the system.

This helps practice removing objects from a list.

---

### 6. Exit Program

The user can safely close the program from the menu.
