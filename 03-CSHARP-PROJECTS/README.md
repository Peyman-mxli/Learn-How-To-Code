This section contains real-world C# projects that I build while learning software development.

The goal of these projects is to move beyond theory and practice solving real problems with code.

While the course modules teach individual concepts, the projects in this section combine many concepts together into complete applications.

Here I learn how professional software is structured, how features work together, and how to turn programming knowledge into usable programs.
