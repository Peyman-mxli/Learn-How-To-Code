🧵 Module — Control Flow in Python
-------------------------------------------------------------
✨ Overview
Control flow is the backbone of every Python program.
It defines how code executes, when decisions are made, and how repetition, errors, and logic paths are handled.

In this module, I study Python’s control flow features from basic decision-making to advanced flow constructs used in professional, real-world codebases.

Mastering control flow allows me to:
Write dynamic, intelligent programs
Handle user input safely
Build robust logic that reacts to data
Structure programs clearly and maintainably
---------------------------------------------------------------
🎯 Learning Objectives
By the end of this module, I will be able to:

Control execution using conditional statements (if, elif, else)
Repeat logic using loops (for, while)
Manage loop behavior with break, continue, and pass
Handle runtime errors using exceptions (try / except / finally)
Use context managers (with) for safe resource handling
Control function execution using return, yield, and raise
Apply modern pattern matching with match / case
Write concise logic using comprehensions and ternary expressions
