🧵 Module — Control Flow — For-loops
✨ Overview
`for` loops in Python are used to **iterate over sequences** such as lists,
tuples, strings, dictionaries, sets, and ranges.
They allow us to repeat actions in a **clear, readable, and Pythonic way**
without manually managing counters like in some other languages.
Python `for` loops are built on the concept of **iterables** and are one of
the most commonly used control-flow tools in real-world programs.
