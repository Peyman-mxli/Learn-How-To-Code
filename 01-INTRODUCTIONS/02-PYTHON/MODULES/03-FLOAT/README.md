🧵 Module — Basic Data Types: Float(flo)
----------------------------------------------
✨ Overview

In this module I focus on Python’s float type — numbers with decimals, like 3.14 or -0.5.
Floats are used everywhere: prices, measurements, grades, statistics, and many real-world calculations.

By the end of this module, I should feel comfortable creating, using, and reasoning about floating-point numbers in real code.

🎯 Key Learning Objectives
By completing this module, I will be able to:

Understand what a float is and when to use it instead of an integer.
Create float variables and use them in expressions.
Perform common arithmetic operations with floats.
Control how floats are displayed (rounding, formatting, etc.).
Be aware of floating-point precision issues and how to deal with them.
Write small, realistic examples that work with prices, averages, and measurements.
