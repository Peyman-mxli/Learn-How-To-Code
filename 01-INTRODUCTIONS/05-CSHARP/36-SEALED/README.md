# 🧵 Module — Sealed

## ✨ Overview

This module focuses on the `sealed` keyword in C#.

The `sealed` keyword is used to prevent inheritance.

When a class is sealed, no other class can inherit from it.

This helps protect important classes from being changed or extended in ways that could break the design of a program.

Sealed is useful when I want to create final classes, protect security-related logic, improve design control, or prevent unsafe inheritance.

Understanding sealed helps me write safer, cleaner, and more professional object-oriented C# code.

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Use sealed methods.
