# 🧵 Module — Attributes

## ✨ Overview

This module focuses on **Attributes** in C#.

Attributes allow me to add extra information, also called metadata, to my code.

I can attach attributes to classes, methods, properties, fields, parameters, and more.

Attributes do not usually change the main logic directly, but they give extra instructions or information that tools, frameworks, compilers, and libraries can read.

Attributes are used in real C# projects for serialization, validation, testing, documentation, APIs, databases, and custom frameworks.

Understanding attributes helps me write more professional C# code and understand how many modern .NET tools work behind the scenes.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Understand what attributes are.
* Understand what metadata means.
* Use built-in C# attributes.
* Apply attributes to classes, methods, and properties.
* Understand attribute syntax.
* Create custom attributes.
* Read attributes using reflection.
* Understand how attributes are used in real projects.
* Recognize why frameworks use attributes heavily.
