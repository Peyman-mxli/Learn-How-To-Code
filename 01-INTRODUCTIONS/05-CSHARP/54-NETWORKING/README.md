# 🧵 Networking

## ✨ Overview

In this module, I learn about **Networking** in C#.

Networking means allowing applications to communicate with other computers, servers, APIs, websites, databases, and services through a network.

Many modern applications use networking every day.

For example, an application may need to:

* Send data to a web API
* Receive data from a server
* Download information from the internet
* Connect to online services
* Communicate between client and server
* Send HTTP requests
* Read HTTP responses

Networking is important because most professional applications are connected to something outside the local computer.

Understanding networking helps me build applications that can communicate with real-world services and exchange data correctly.
