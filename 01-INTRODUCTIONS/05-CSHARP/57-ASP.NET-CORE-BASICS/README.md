# 🧵 ASP.NET Core Basics

## ✨ Overview

In this module, I learn the basics of ASP.NET Core.

ASP.NET Core is a modern framework from Microsoft used to build web applications, websites, backend systems, and APIs using C#.

With ASP.NET Core, I can create applications that run on:

- Windows
- Linux
- macOS
- Cloud servers
- Web browsers
- Mobile apps through APIs

ASP.NET Core is very important because many professional C# developers use it to build real-world web systems.

Examples include:

- School systems
- Store systems
- Login systems
- Dashboards
- REST APIs
- Admin panels
- Business applications

This module helps me understand how C# moves from Console Applications into real web development.

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

- Understand what ASP.NET Core is
- Understand what a web application is
- Understand the difference between frontend and backend
- Understand controllers

- Understand views

- Understand middleware
- Understand dependency injection in ASP.NET Core
- Understand request and response flow
- Create simple web endpoints

- Prepare for professional web development with C#
