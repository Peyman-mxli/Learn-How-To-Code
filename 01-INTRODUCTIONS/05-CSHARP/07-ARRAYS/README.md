# 🧵 - Arrays

## Introduction

In this module, I learn how to store multiple values inside one variable structure in C#.

An array is a collection that can hold many values of the same data type.

Instead of creating many separate variables, I can use one array to organize related data.

For example, an array can store:

* Student names
* Scores
* Item prices
* Game levels
* Player inventory items

---

# Why Arrays Matter

Arrays are important because they help organize multiple values efficiently.

With arrays, I can:

* Store lists of data
* Avoid repeated variables
* Process many values with loops
* Build inventory systems
* Store scores and grades
* Prepare for collections and lists later

Arrays make programs cleaner, more organized, and easier to manage.
