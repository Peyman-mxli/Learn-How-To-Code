# 🧵 - Loops

## Introduction

In this module, I learn how to repeat code in C#.

Loops allow a program to run the same block of code multiple times without writing the same code again and again.

This is important because many programs need repetition.

For example, a program can repeat code to:

* Count numbers
* Display menus
* Process lists of data
* Repeat a game action
* Ask for input until it is correct
* Run calculations multiple times

---

# What I Learn In This Module

### For Loop

I learn how to repeat code a specific number of times.

### While Loop

I learn how to repeat code while a condition is true.

### Do While Loop

I learn how to run code at least once before checking the condition.

### Loop Counters

I learn how counters control how many times a loop runs.

### Break Statement

I learn how to stop a loop early.

### Continue Statement

I learn how to skip part of a loop and continue with the next repetition.

### Nested Loops

I learn how to place one loop inside another loop.

### Infinite Loops

I learn what infinite loops are and why I must avoid them.

---

# Why Loops Matter

Loops are one of the most important parts of programming.

With loops, I can:

* Repeat actions automatically
* Reduce duplicated code
* Build menus
* Process multiple values
* Create simple games
* Work with arrays and collections later
* Make programs more powerful and efficient

Without loops, I would need to write the same code many times manually.
