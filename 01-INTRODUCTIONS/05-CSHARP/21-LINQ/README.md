# 🧵 - LINQ

## Introduction

In this module, I learn about **LINQ**, which means **Language Integrated Query**.

LINQ allows me to search, filter, sort, group, and transform data directly inside C#.

Instead of writing long and complicated loops, LINQ helps me work with collections in a cleaner and more professional way.

LINQ is commonly used with arrays, lists, databases, objects, XML, and many real-world data systems.

Understanding LINQ helps me write shorter, cleaner, and more powerful C# code.

## Simple Example

LINQ can filter numbers from a list.

```csharp
List<int> numbers = new List<int> { 1, 2, 3, 4, 5, 6 };

var evenNumbers = numbers.Where(number => number % 2 == 0);

foreach (int number in evenNumbers)
{
    Console.WriteLine(number);
}
```

This finds only the even numbers from the list.

## Real-World Example

Imagine I have a list of students.

Each student has:

* IsActive status

With LINQ, I can easily find:

* Students with high grades
* Active students only
* Students sorted by name
* Students grouped by grade

LINQ makes data filtering and organization much easier.
