# 🧵 - Conditions

## Introduction

In this module, I learn how to make decisions in C#.

Conditions allow a program to choose what code should run based on whether something is true or false.

This is important because real programs need to respond differently depending on different situations.

For example, a program can check:

* If a user is old enough
* If a password is correct
* If a player has enough health
* If a grade is passing
* If a number is positive or negative

---

# What I Learn In This Module

### If Statements

I learn how to run code only when a condition is true.

### Else Statements

I learn how to run different code when a condition is false.

### Else If Statements

I learn how to check multiple conditions in order.

### Boolean Conditions

I learn how true and false values control decisions.

### Comparison Operators in Conditions

I learn how to use operators like:

```csharp
== != > < >= <=
```

### Logical Operators in Conditions

I learn how to combine conditions using:

```csharp
&& || !
```

### Nested Conditions

I learn how to place one condition inside another condition.

### Real-World Decision Making

I learn how conditions are used in games, apps, login systems, menus, and business software.

---

# Why Conditions Matter

Conditions are one of the most important parts of programming.

With conditions, I can create programs that:

* Make decisions
* React to user input
* Validate information
* Control access
* Check game states
* Display different messages
* Run different actions based on rules

Without conditions, programs would always run the same way every time.
