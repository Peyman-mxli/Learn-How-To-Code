# 🧵 Reflection

## ✨ Overview

In this module, I learn about Reflection in C#.

Reflection allows me to inspect information about classes, methods, properties, fields, constructors, attributes, and assemblies while the program is running.

Normally, I write code that already knows which classes and methods it will use. With Reflection, I can discover that information dynamically at runtime.

Reflection is a powerful feature used in frameworks, dependency injection containers, ORMs, testing tools, plugins, serialization systems, and many advanced .NET applications.

Because Reflection works at runtime, it gives me flexibility, but it can also affect performance if used incorrectly.

Learning Reflection helps me understand how many professional C# frameworks work behind the scenes.
