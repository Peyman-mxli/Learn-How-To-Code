# 🧵 Deserialization

## ✨ Overview

In this module, I learn about Deserialization in C#.

Deserialization means converting stored data back into a real C# object.

If serialization is:

Object → JSON

Then deserialization is:

JSON → Object

This is important because real applications often need to read saved files, load settings, restore user data, receive API responses, and rebuild objects from stored information.

Deserialization helps my program understand external data and use it like normal C# objects.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

- Understand what deserialization means
- Convert JSON text into C# objects
- Use `JsonSerializer.Deserialize<T>()`
- Read JSON data from files
- Restore objects from saved data
- Deserialize lists and arrays
- Handle missing or extra JSON properties
- Understand common deserialization problems
- Learn safe deserialization practices
