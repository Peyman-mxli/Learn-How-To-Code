# 🧵 Entity Framework

## ✨ Overview

In this module, I learn about Entity Framework, one of the most important technologies in modern C# application development.

Entity Framework (EF) is Microsoft's Object-Relational Mapper (ORM).

An ORM allows me to work with databases using C# objects instead of writing large amounts of SQL manually.

Instead of creating SQL queries for every operation, I can create classes that represent database tables and let Entity Framework handle the communication between my application and the database.

Entity Framework is heavily used in:

- ASP.NET Core applications
- Enterprise software
- APIs
- Desktop applications
- Cloud applications
- Professional business systems

Learning Entity Framework is a major step toward becoming a professional .NET developer.
