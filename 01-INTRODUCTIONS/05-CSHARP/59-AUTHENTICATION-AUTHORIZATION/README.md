# 🧵 Authentication Authorization

## ✨ Overview

In this module, I learn about Authentication and Authorization in C# and ASP.NET Core.

Authentication and Authorization are very important for building secure applications.

Authentication answers this question:

```text
Who are you?
```

Authorization answers this question:

```text
What are you allowed to do?
```

For example:

A user may log in successfully, but that does not mean the user can access every part of the system.

A normal user may view their profile.

An admin may manage users.

A manager may view reports.

Security is one of the most important parts of professional software development.
