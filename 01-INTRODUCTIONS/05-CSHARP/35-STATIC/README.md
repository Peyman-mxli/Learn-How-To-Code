# 🧵 Module — Static

## ✨ Overview

This module focuses on the `static` keyword in C#.

Static members belong to the class itself, not to a specific object created from the class.

This means I can use a static field, method, property, or class without creating an object first.

Static is useful when I need shared data, utility methods, counters, configuration values, helper classes, or global-style behavior inside a controlled class structure.

Understanding static helps me write cleaner, more organized, and more professional C# code.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Understand what `static` means in C#.
* Create static fields.
* Create static methods.
* Create static properties.
* Understand how static members are shared.
* Understand the difference between instance members and static members.
* Create static classes.
* Use utility/helper classes correctly.
* Avoid common mistakes with static data.
