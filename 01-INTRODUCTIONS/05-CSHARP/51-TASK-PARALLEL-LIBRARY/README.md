# 🧵 Task Parallel Library

## ✨ Overview

In this module, I learn about the **Task Parallel Library (TPL)** in C#.

The Task Parallel Library is a powerful part of .NET that helps me run work asynchronously and in parallel.

Instead of manually creating and managing threads, I can use tasks to make my code cleaner, safer, and easier to understand.

Tasks are commonly used when I want to:

* Run background work
* Execute multiple operations at the same time
* Improve performance
* Keep applications responsive
* Handle asynchronous programming professionally

TPL is important because modern C# applications often need to process data, call APIs, load files, perform calculations, or run background operations without blocking the main program.

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Understand what the Task Parallel Library is
* Understand what a Task is

* Use Task.Run()
* Wait for tasks to complete
* Return values from tasks
* Run multiple tasks at the same time
* Use Task.WaitAll()
* Use Task.WhenAll()
* Understand parallel execution
* Learn basic task exception handling
* Understand why TPL is preferred over manual thread management
