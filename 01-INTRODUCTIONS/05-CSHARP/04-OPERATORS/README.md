# 04 - Operators

## Introduction

In this module, I learn how operators work in C#.

Operators are special symbols used to perform actions on values and variables.

They allow me to calculate numbers, compare values, assign data, and build conditions inside programs.

Operators are important because they help programs make decisions and process information.

---

# What I Learn In This Module

### Arithmetic Operators

I learn how to perform basic math operations such as:

* Addition
* Subtraction
* Multiplication
* Division
* Modulus

### Assignment Operators

I learn how to assign and update values inside variables.

### Comparison Operators

I learn how to compare values and return true or false results.

### Logical Operators

I learn how to combine multiple conditions using logical operators.

### Increment and Decrement Operators

I learn how to increase or decrease values by one.

### Operator Precedence

I learn how C# decides which operation happens first.

---

# Why Operators Matter

Operators are used in almost every program.

With operators, I can:

* Perform calculations
* Compare user input
* Check conditions
* Update scores
* Build menus
* Create game logic
* Control program behavior

Understanding operators helps me write programs that can think, calculate, and make decisions.
