# 03 - Input & Output

## Introduction

In this module, I learn how a C# program communicates with the user.

Input means receiving information from the user.

Output means showing information to the user.

In Console Applications, I usually use the console window to display messages and receive keyboard input.

This is important because many programs need to interact with users instead of only running fixed instructions.

---

# What I Learn In This Module

### Output

I learn how to display text using:

```csharp
Console.WriteLine();
Console.Write();
```

### Input

I learn how to receive text from the user using:

```csharp
Console.ReadLine();
```

### Reading User Information

I learn how to ask the user for information such as:

* Name
* Age
* City
* Favorite language

### Combining Input and Output

I learn how to take user input and display a personalized message.

### Converting Input

I learn that input from `Console.ReadLine()` is received as text, and numbers must be converted before using them as numbers.

---

# Why Input & Output Matters

Input and output are the foundation of interactive programs.

With input and output, I can create programs that:

* Ask questions
* Display results
* Respond to users
* Create simple menus
* Build beginner console projects
