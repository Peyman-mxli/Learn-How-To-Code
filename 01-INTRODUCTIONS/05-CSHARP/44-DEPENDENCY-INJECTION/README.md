# 🧵 Dependency Injection

## ✨ Overview

In this module, I learn about Dependency Injection in C#.

Dependency Injection means giving a class the objects it needs instead of creating them directly inside the class.

A dependency is something a class needs to do its work.

For example, a service may need:

- A database repository
- A logger
- An email sender
- A file manager
- A payment processor
- A configuration object

Instead of creating those objects inside the class, I inject them from the outside.

Dependency Injection is important because it helps me write cleaner, more flexible, and more testable code.
