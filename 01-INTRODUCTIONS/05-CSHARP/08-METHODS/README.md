# 🧵 - Methods

## Introduction

In this module, I learn how to organize code using methods in C#.

A method is a reusable block of code that performs a specific task.

Instead of writing the same code many times, I can place that code inside a method and call it whenever I need it.

Methods help make programs cleaner, easier to read, easier to test, and easier to maintain.

---

# What I Learn In This Module

### What Methods Are

I learn how methods group code into reusable sections.

### Creating Methods

I learn how to create my own methods in C#.

### Calling Methods

I learn how to run a method from another part of the program.

### Void Methods

I learn how methods can perform actions without returning a value.

### Methods With Parameters

I learn how to send information into a method.

### Methods With Return Values

I learn how methods can calculate and return results.

### Method Naming

I learn how to use clear and meaningful method names.

### Reusable Code

I learn how methods help avoid repeated code.

---

# Why Methods Matter

Methods are important because they help organize programs.

With methods, I can:

* Reuse code
* Reduce duplication
* Organize large programs
* Make code easier to read
* Separate tasks into smaller parts
* Build cleaner projects
* Prepare for object-oriented programming

Without methods, programs can become long, messy, and difficult to understand.
