# 🧵 - Constructors

## Introduction

In this module, I learn how constructors work in C#.

Constructors are special methods that automatically run when an object is created.

They are used to initialize object data and prepare objects for use.

Constructors are one of the most important features of Object-Oriented Programming because they allow objects to start with meaningful values immediately after creation.

# Why Constructors Matter

Constructors are important because they help ensure objects start in a valid state.

With constructors, I can:

* Initialize data automatically
* Reduce repetitive assignments
* Improve code readability
* Create cleaner object creation patterns
* Build more reliable classes
* Prepare for advanced Object-Oriented Programming concepts

Without constructors, I often need to assign values manually after creating every object.

---

# Real-World Example

Without Constructor:

```csharp
Player hero = new Player();

hero.playerName = "Peyman";
hero.level = 25;
hero.health = 100;
```

With Constructor:

```csharp
Player hero = new Player("Peyman", 25, 100);
```

The constructor automatically assigns the values.
