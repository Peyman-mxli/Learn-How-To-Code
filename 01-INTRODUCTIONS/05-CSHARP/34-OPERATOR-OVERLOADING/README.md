# 🧵 Module — Operator Overloading

## ✨ Overview

This module focuses on **Operator Overloading** in C#.

Operator overloading allows me to define how operators such as `+`, `-`, `*`, `/`, `==`, and `!=` behave when used with my own custom classes and structs.

This feature makes custom objects feel more natural to use and can greatly improve code readability when applied correctly.

Operator overloading is commonly used in applications that work with mathematical objects, vectors, currencies, measurements, game statistics, and other custom data types.

Understanding operator overloading helps me write cleaner, more expressive, and more professional C# code.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Understand what operator overloading is.
* Understand why operator overloading exists.
* Create overloaded arithmetic operators.
* Create overloaded comparison operators.
* Use the `operator` keyword correctly.
* Understand why overloaded operators must be `static`.
* Improve the readability of custom classes.
* Recognize when operator overloading is useful and when it should be avoided.
* Build more professional and intuitive APIs.
