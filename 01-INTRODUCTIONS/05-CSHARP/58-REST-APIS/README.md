# 🧵 REST APIs

## ✨ Overview

In this module, I learn about REST APIs in C# and ASP.NET Core.

A REST API allows different applications to communicate with each other through HTTP.

For example, a website, mobile app, or desktop app can send requests to a backend server and receive data in return.

REST APIs are used in many real-world systems such as:

- Login systems
- Online stores
- Student systems
- Banking apps
- Social media apps
- Inventory systems
- Mobile applications

Learning REST APIs is very important because most modern software uses APIs to send and receive data.
