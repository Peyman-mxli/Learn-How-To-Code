# 🧵 Serialization

## ✨ Overview

In this module, I learn about Serialization in C#.

Serialization means converting an object into a format that can be saved, stored, transferred, or sent to another system.

Deserialization means converting that saved data back into an object.

This is important because real programs often need to save data, load data, send data through APIs, store settings, or work with files.

Serialization is commonly used with formats like JSON and XML.

In modern C# applications, JSON serialization is one of the most common ways to store and exchange data.

- Understand what deserialization is
