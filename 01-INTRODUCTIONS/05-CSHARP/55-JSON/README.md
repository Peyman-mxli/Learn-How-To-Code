# 🧵 JSON

## ✨ Overview

In this module, I learn about **JSON** in C#.

JSON means **JavaScript Object Notation**.

JSON is a lightweight text format used to store and exchange data between applications.

Many modern systems use JSON because it is simple, readable, and easy for both humans and computers to understand.

For example, JSON is commonly used in:

* Web APIs
* Configuration files
* Mobile applications
* Web applications
* Databases
* Game data
* Cloud services

When I understand JSON, I can send data, receive data, save data, load data, and connect my C# applications to real-world services.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

* Understand what JSON is
* Understand why JSON is important
* Learn JSON syntax
* Understand JSON objects
* Understand JSON arrays
* Understand key/value pairs
* Understand how JSON represents strings, numbers, booleans, and null
* Serialize C# objects into JSON
* Deserialize JSON into C# objects
* Use `System.Text.Json`
* Read JSON data
* Write JSON data
* Understand JSON best practices
