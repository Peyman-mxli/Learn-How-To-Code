# 📘 Module — Basic Data Types: Double (`double`)

This module introduces one of the most important decimal number data types in C#: `double`.

Here, I learn how C# stores decimal numbers, how double variables work, and how decimal values behave when I use them in real code.

Doubles are used when a number needs decimal places, such as prices, measurements, grades, percentages, speed, distance, tax rates, and calculations that are not whole numbers.

Understanding doubles is important because many real-world programs need accurate decimal-based calculations.

In this module, I work on:

🧠 Understanding what a `double` is

➖ Subtracting decimal numbers  
✖️ Multiplying decimal numbers  
➗ Dividing decimal numbers  
💰 Using doubles for prices and totals
