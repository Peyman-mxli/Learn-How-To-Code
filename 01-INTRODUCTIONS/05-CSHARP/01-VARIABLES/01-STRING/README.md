# 📘 Module — Basic Data Types: String (`string`)

This module introduces one of the most important data types in C#: `string`.

Here, I learn how C# stores text, how string variables work, and how text values behave when I use them in real code.

Strings are used everywhere in programming: names, messages, passwords, menus, labels, file paths, user input, and text output.

Understanding strings well is essential before moving on to more advanced topics.

---

## 📚 What I Learn in This Module

In this module, I work on:

🧠 Understanding what a `string` is  
🔤 Creating string variables  
💬 Printing text to the console  
➕ Combining strings together  
🧩 Using string interpolation with `$""`  
🔢 Checking string length  
🔍 Accessing characters inside a string  
✂️ Using common string methods  
🧼 Cleaning user input with `Trim()`  
🔠 Changing text with `ToUpper()` and `ToLower()`  
🧪 Practicing realistic string examples  
⚠️ Understanding common beginner mistakes with strings
