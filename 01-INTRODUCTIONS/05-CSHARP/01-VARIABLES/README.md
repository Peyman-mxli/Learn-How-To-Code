# 🧵 Module 01 — Variables & Basic Data Types in C#

This module introduces the foundations of C# variables. Here, I learn how C# stores information, how variables work, and how different basic data types behave when I use them in real code.

Variables are the building blocks 🧱 of every C# program. Understanding them well is essential before moving on to more advanced topics like conditions, loops, methods, classes, and object-oriented programming.

---

## 📚 What I Learn in This Module

In this module, I work on:

🧠 Understanding what a variable is and how to create one
🔤 Learning how to store text using `string`
🔢 Learning how to store whole numbers using `int`
💰 Learning how to store decimal numbers using `double`
✅ Learning how to store true or false values using `bool`
✍️ Writing clear and readable variable names
🧩 Practicing how variables are used in real C# programs
⚠️ Understanding common beginner mistakes with variables
🏗️ Building strong foundations for future C# topics

---

## 📁 Folders Included in This Module

### 📄 01-STRING

This folder focuses on the `string` data type.

I learn how C# stores text, names, messages, sentences, and user input.

### 📄 02-INTEGER

This folder focuses on the `int` data type.

I learn how C# stores whole numbers such as age, score, quantity, and count.

### 📄 03-DOUBLE

This folder focuses on the `double` data type.

I learn how C# stores decimal numbers such as prices, grades, measurements, and calculations.

### 📄 04-BOOLEAN

This folder focuses on the `bool` data type.

I learn how C# stores logical values: `true` and `false`.

---

## 📂 Files Included in Each Folder

Each data type folder follows the same structure:

📄 `Notes.cs`
Detailed explanations of each concept with short, simple examples and comments.

📝 `Tasks.cs`

Clean, well-written solutions that I can compare against my work to improve my coding style.
