# 📘 Module — Basic Data Types: Integer (`int`)

This module introduces one of the most common number data types in C#: `int`.

Here, I learn how C# stores whole numbers, how integer variables work, and how numbers behave when I use them in real code.

Integers are used everywhere in programming: age, score, level, quantity, count, money, health points, inventory slots, and many other values that do not need decimals.

Understanding integers well is essential before moving on to calculations, conditions, loops, arrays, methods, and real applications.

---

## 📚 What I Learn in This Module

In this module, I work on:

🧠 Understanding what an `int` is  
🔢 Creating integer variables  
🖨️ Printing integer values  
➕ Adding numbers  
➖ Subtracting numbers  
✖️ Multiplying numbers  
➗ Dividing numbers  
🔁 Reassigning integer values  
🎮 Using integers for scores, levels, health, and counts  
⚠️ Understanding common beginner mistakes with integers  
🏗️ Practicing realistic integer examples
