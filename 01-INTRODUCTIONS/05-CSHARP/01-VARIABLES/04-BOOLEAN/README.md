# 📘 Module — Basic Data Types: Boolean (`bool`)

This module introduces one of the most important logic data types in C#: `bool`.

Here, I learn how C# stores true or false values, how boolean variables work, and how logical values behave when I use them in real code.

Booleans are used everywhere in programming: checking if a user is logged in, if a player is alive, if a password is valid, if a file exists, if a button was clicked, or if a condition is true.

Understanding booleans is essential before moving on to conditions, loops, validation, methods, classes, and real applications.

---

## 📚 What I Learn in This Module

In this module, I work on:

🧠 Understanding what a `bool` is  
✅ Creating boolean variables  
🔁 Storing `true` and `false` values  
🖨️ Printing boolean values  
⚖️ Understanding comparison results  
🔐 Using booleans for validation  
🎮 Using booleans for game states  
🚪 Using booleans for open / closed states  
🧪 Practicing realistic boolean examples  
⚠️ Understanding common beginner mistakes with booleans  
🏗️ Building strong logic foundations for future C# topics
