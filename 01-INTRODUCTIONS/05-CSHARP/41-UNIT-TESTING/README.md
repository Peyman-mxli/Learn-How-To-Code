# 🧵 Unit Testing

## ✨ Overview

In this module, I learn about Unit Testing in C#.

Unit testing means testing small parts of my code separately to make sure each part works correctly.

A unit can be:

- A method
- A class
- A small piece of logic

Unit testing is important because it helps me find problems early, protect my code from future bugs, and write more reliable software.

In professional C# development, unit testing is commonly used with tools like xUnit, NUnit, and MSTest.

---

## 🎯 What I will learn in this module

By the end of this module, I will be able to:

- Understand what unit testing is
- Understand why testing is important
- Learn what a unit test checks
- Learn the Arrange, Act, Assert pattern
- Understand expected results
- Write simple testable methods
- Understand basic test cases
- Learn the difference between passing and failing tests
- Understand how unit testing improves code quality
