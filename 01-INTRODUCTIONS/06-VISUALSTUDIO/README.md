 #  VISUAL STUDIO

# What is Visual Studio?

Visual Studio is an IDE (Integrated Development Environment) made by Microsoft.

It helps developers create applications using:

* C#
* .NET
* C++
* Python
* Web Development

In this course we will use Visual Studio to learn:

* C#
* .NET
* Console Applications

---

# Download Visual Studio

Go to the official Microsoft website:

https://visualstudio.microsoft.com/downloads/

Download:

```text
Visual Studio Community
```

This version is:

* Free
* Beginner friendly
* Perfect for learning C# and .NET

---

# Install Visual Studio

Open the installer.

When the workloads screen appears, select:

```text
.NET desktop development
```

Then click:

```text
Install
```

Wait for the installation to finish.

---

# Create Your First Project

Open Visual Studio.

Click:

```text
Create a new project
```

Search for:

```text
Console App
```

Select:

```text
Console App
```

IMPORTANT:

Use:

```text
Console App
```

Do NOT use:

```text
Console App (.NET Framework)
```

The normal "Console App" uses modern .NET.

---

# Create the Project

1. Click Next
2. Name the project
3. Click Create

Visual Studio will generate the project automatically.

---

# First C# Program

Open:

```text
Program.cs
```

Example:

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello World");
    }
}
```

---

# Run the Program

Press:

```text
Ctrl + F5
```

Or click the green Start button.

Output:

```text
Hello World
```
