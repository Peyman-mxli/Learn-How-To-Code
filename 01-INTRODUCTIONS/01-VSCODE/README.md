📌 What Is Visual Studio Code?
Visual Studio Code (VS Code) is a free, lightweight, and powerful code editor developed by Microsoft.

It is used to:

Write code
Run Python programs
Debug errors
Organize projects
Install extensions (plugins)
⚠️ VS Code is NOT a programming language.

🌟 Why Use VS Code for Python?
VS Code is popular because:

Easy to use for beginners
Extremely powerful for professionals
Supports Python very well
Works on Windows, macOS, and Linux
Has thousands of extensions
Many developers and companies use VS Code daily.
